Welcome to tk-katana’s documentation!¶

A Shotgun Engine for Katana

This engine provides Shotgun Toolkit integration for The Foundry’s Katana.

This package was re-created with Cookiecutter and the audreyr/cookiecutter-pypackage project template.

Forked from robblau’s v0.1.0 to hopefully make it production capable again for:

  • Shotgun 8 and above
  • Katana 3.1 (PyQt5) using Qt.py 1.1.0
